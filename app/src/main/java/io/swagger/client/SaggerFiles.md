# Guide to Swagger Files

## api

Files for connection to the API
NOTE: do not manually modify these files - they are generated by swagger

## auth

Files to provide structure for authentication with server
NOTE: do not manually modify these files - they are generated by swagger

## model

Files to provide structures for objects to go to and from the server
NOTE: these files should not be manually modified, although they can implement simple interfaces like "serializable" as needed

## Other

Files to perform various other operations in connection with the server. Most of these should not be manually modified, although ApiClient.java must be updated to reflect the server location
